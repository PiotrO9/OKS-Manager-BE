# BUSINESS RULES

1. Student:

- can have multiple courses
- can book lessons

2. Instructor:

- has availability
- can be assigned to multiple schools

3. Lesson:

- must belong to a course
- must respect availability
- must respect OSK consistency

4. Rating:

- only after completed practice lesson
- one rating per lesson

5. Payments:

- course has payment plan
- payments can be full or installments
